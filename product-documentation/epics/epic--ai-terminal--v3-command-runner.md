# Epic: AI Terminal V3 - Command Runner

**Product:** [AI Terminal](../../product-vision--ai-terminal.md)
**Version / Release Label:** V3 - Command Runner

## Goal Statement
Deliver Command Runner integration services of the AI Terminal app, enabling a single click-command to copy from the AI Chat, paste into the Terminal prompt, and automatically submit the text as a command, as well as copying text from the terminal, pasting into the AI Chat input and submitting it for analysis.

## Success Criteria
*Each criterion must trace to a KPI or OKR defined in the product vision.*

- [Chat-to-Terminal Speed] — advances this metric by providing single-click functionality to run text in an AI chat as a command in the terminal.
- [Terminal-Result-to-Chat-Analysis Speed] — advances this metric by providing single-click functionality to submit text from the terminal to AI for analysis.
