# Epic: AI Terminal V2 - Terminal Management

**Product:** [AI Terminal](../../product-vision--ai-terminal.md)
**Version / Release Label:** V2 - Terminal Management

## Goal Statement
Deliver the terminal management capabilities of the AI Terminal app, enabling technical users to add, open, and manage one or more terminal sessions within a single workspace, including manually copying highlighted text from the terminal and pasting clipboard text into the terminal prompt.

## Success Criteria
*Each criterion must trace to a KPI or OKR defined in the product vision.*

- [Terminal-Result-to-Chat-Analysis Speed] — advances this metric by facilitating integrated terminals that can directly copy highlighted terminal text for transfer to the AI chat via the clipboard.
